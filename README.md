# image2ascii

Converts an image to an ASCII image.

```shell
> python3 ascii.py [options]
```

options:  
**-h, --help:**         Prints help message.  
**--path p:**           Path to image.  
**--width w:**          Sets ASCII image width to 'w'. Default: '100'.  
**--invert:**           Inverts ASCII greyscale.
